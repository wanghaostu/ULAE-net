# ULAE-net
code for  "Medical Image Registration Based on Uncoupled Learning and Accumulative Enhancement".


# Prerequisites
Python 3.5.2+

Pytorch 1.3.0 - 1.7.0

SimpleITK

This code has been tested with Pytorch 1.3.0 and GTX3080 GPU.
